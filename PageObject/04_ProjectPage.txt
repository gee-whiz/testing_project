#******************** Test Cases *************************

*** Test Case ***

# Positive Tests

Create Blank Project
    Navigate To Projects Page  ProjectsUrl
    Click New Project
    Select Project Type  TemplateType
    Input Project Name  BlankProjectName
    Input Project Description  Description
    Set Visibility  Visibility
    Create Project
    Verify Page Should Contain Project 'BlankProjectName' was successfully created
    Verify Project Overview Shows Name  BlankProjectName
    
Create Template Project
    Navigate To Projects Page  ProjectsUrl
    Click New Project
    Select Create From Template
    Choose Template  TemplateType
    Input Project Name  TemplateProjectName
    Set Visibility  Visibility
    Create Project
    Verify Page Should Contain Project 'TemplateProjectName' was successfully created
    Verify Project Files Are Present From Template
    
Update Name Of Template Project
    Navigate To Project  TemplateProjectName
    Open Project Settings General
    Edit Project Name  RenamedTemplateProjectName
    Save Project Settings
    Verify Page Should Contain Successfully updated project settings
    Verify Project Name Displays  RenamedTemplateProjectName
    
Delete Template Project
    Navigate To Project  RenamedTemplateProjectName
    Open Project Settings Advanced
    Initiate Project Deletion
    Confirm Deletion By Typing Project Name  RenamedTemplateProjectName
    Confirm Delete Project
    Verify Page Should Contain Project deleted successfully
    Verify Project Not Listed In Projects Page  RenamedTemplateProjectName
    
Validate Success Message On Create
    Navigate To Projects Page  ProjectsUrl
    Click New Project
    Select Project Type  TemplateType
    Input Project Name  BlankProjectName
    Create Project
    Verify Page Should Contain Project 'BlankProjectName' was successfully created

# Negative Tests

Create A Blank Project With The Previous Same Name
    Navigate To Projects Page  ProjectsUrl
    Click New Project
    Select Project Type  TemplateType
    Input Project Name  DuplicateProjectName
    Create Project
    Verify Page Should Contain Name has already been taken
    Verify Project Not Created With Duplicate Name
