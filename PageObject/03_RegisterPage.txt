#******************** Test Cases *************************

*** Test Case ***

# Positive Tests

Register User With Valid Username And Password
    Navigate To Register Page  LoginUrl
    Input Username  NewUserName
    Input Email  NewEmail
    Input Password  NewPassword
    Confirm Password  ConfirmPassword
    Complete Recaptcha
    Submit Registration Form
    Verify Page Should Contain confirmation instructions for email verification
    Verify Login Is Possible With New Credentials After Confirmation

# Negative Tests
	
Register User With Same Above Username
    Navigate To Register Page  LoginUrl
    Input Username  NewUserName
    Input Email  NewEmail
    Input Password  NewPassword
    Confirm Password  ConfirmPassword
    Complete Recaptcha
    Submit Registration Form
    Verify Page Should Contain Username has already been taken
    Verify Registration Form Remains Visible
    
Register User With Empty Password
    Navigate To Register Page  LoginUrl
    Input Username  NewUserName
    Input Email  NewEmail
    Input Password  Empty
    Confirm Password  Empty
    Complete Recaptcha
    Submit Registration Form
    Verify Page Should Contain This field is required
    
Register User With Invalid Email
    Navigate To Register Page  LoginUrl
    Input Username  NewUserName
    Input Email  InvalidEmail
    Input Password  NewPassword
    Confirm Password  ConfirmPassword
    Complete Recaptcha
    Submit Registration Form
    Verify Page Should Contain Email is invalid
    
Register User Without Recaptcha
    Navigate To Register Page  LoginUrl
    Input Username  NewUserName
    Input Email  NewEmail
    Input Password  NewPassword
    Confirm Password  ConfirmPassword
    Skip Recaptcha
    Submit Registration Form
    Verify Page Should Contain Please verify that you are not a robot
